## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Versions

```bash
"@nestjs/typeorm": "7.1.5",
"typeorm": "0.2.31"
"body-parser": "^1.20.1",
```

## Installation

```bash
$ npm install
```

## Install Dependencies

```bash
$ yarn add bcrypt @nestjs/config @nestjs/event-emitter @nestjs/jwt passport @nestjs/passport @nestjs/platform-express typeorm @nestjs/typeorm pg passport-jwt passport-local swagger-ui-express moment class-transformer class-validator nestjs-xion rand-token @nestjsx/crud @nestjsx/crud-request @nestjsx/crud-typeorm
```

## Install Dev Dependencies

```bash
$ yarn add -D @types/bcrypt @nestjs/swagger @types/passport @types/passport-jwt @types/passport-local husky @types/lodash 
```

## Install existing project

```bash
$ yarn install

```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Command lines
```bash
# After create an entity filel or many files

# update new code
$ yarn build

# generate new migration
$ yarn typeorm migration:generate -n update-table-name (update-table-name => any thing)

# run migration 
$ yarn typeorm migration:run

# revert migration
$ yarn typeorm migration:revert
```
